# Ocean Project — Page d’accueil "Atlantic Ink"

**Description**  
Ce projet est la page d’accueil interactive de l’exploration de l’océan Atlantique.  
Il présente un pré-chargement immersif avec :  
- Fond bleu nuit  
- Ligne ondulée lumineuse animée rappelant la surface de la mer  
- Texte introductif progressif : "L’océan Atlantique respire. Entrez."  
- Ambiance sonore légère (fichier WAV inclus)  
- Bouton "Commencer l’exploration" menant à la future page interactive

**Technologies utilisées**  
- Next.js 14 (App Router)  
- React  
- Framer Motion (animations du texte et de la ligne ondulée)  
- HTML / CSS pour le style  
- Fichier audio WAV intégré dans `public/assets`
